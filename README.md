# vim_config

## 설치방법
다음과 같은 명령어를 사용하여 vim plugin Vundle을 설치한다.  
```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

그후 console 명령창에서  
```
vim +PluginInstall +qall
```
를 눌러서 Plugin들을 자동으로 설치해준다.
